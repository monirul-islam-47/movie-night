# Movie Night VERSION

Movie Night lets a desktop host choose a movie and send one private browser
link. Guests install nothing and create no account.

## Choose your download

- **Windows 10/11, 64-bit:** `WINDOWS_ASSET`
- **Linux x86-64:** `LINUX_APPIMAGE` or portable `LINUX_TAR`
- **macOS:** a native app is not available in this release.

## Before running

These builds are **unsigned**. Windows may display Unknown publisher or
SmartScreen. Download only from the official Movie Night page and compare the
published SHA-256 manifest. Checksums verify file integrity, not publisher
identity.

## What happens next

Open Movie Night, choose or resume a movie, wait for the room to become
watchable, then send the guest invite from inside the room. End with Goodnight
to close the server, preparation, and connection helpers together.

System requirements, exact artifact sizes, checksums, test evidence, and
third-party notices accompany this release.
